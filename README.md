# my-project# Product Inventory Management System

## Project Description
This project is a simple Product Inventory Management System. It helps to manage product details such as product name, quantity, and price.

## Features
- Add new products
- Update product details
- Delete products
- View product list

## Technology Used
- Python
- File Handling / Database

## Purpose
The main purpose of this project is to manage and track products easily in a small inventory system.
